## How to build
```shell
# Change directory
$ cd tex or tex/abst
# Build main file
$ make
# Build main without bibtex
$ make simple
# Build main file only for dvi
$ make dvi
# Clean internal files
$ make clean
# Clean internal files including pdf
$ make realclean
```
